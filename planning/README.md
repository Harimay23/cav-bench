# Planning archive

These are the original v0.3 → v1.0 hardening specification documents,
preserved as-is for provenance. They describe the *plan*; they are not
kept up to date as the implementation evolves.

For current, maintained documentation, see:

- [`../README.md`](../README.md) — what CAV-Bench is, quickstart, results
- [`../docs/`](../docs/) — methodology, architecture, authoring guides, reproducibility
- [`../DECISION_LOG.md`](../DECISION_LOG.md) — the actual decisions made, including deviations from this plan and why
- [`../ACCEPTANCE_CHECKLIST.md`](../ACCEPTANCE_CHECKLIST.md) (repo root) — the maintained release checklist; this directory's copy is the original, frozen version

| File | Role |
|---|---|
| `00_READ_ME_FIRST.md` | Handoff status and reading order |
| `PRD.md` | Product requirements for v1.0 |

| `TECHNICAL_DESIGN.md` | Schemas, APIs, CLI contracts as originally specified |
| `IMPLEMENTATION_PLAN.md` | The phase-by-phase execution plan this project followed |
| `ACCEPTANCE_CHECKLIST.md` | The original release gate (frozen copy) |
